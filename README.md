# Word to HTML Editor | Online Tool + npm Package

Easily convert word into clean, semantic HTML.

This project offers:
- 🌐 A **browser-based editor** to paste content and get instant HTML output.
- 📦 A **Node.js npm package** to integrate editor into your own apps and pipelines.

## ✨ Features
- Converts word to HTML
- Removes unnecessary styles and Word-specific markup
- Supports basic formatting: headings, bold, italic, lists, tables, links, and images
- Clean, readable, semantic HTML output
- Lightweight and dependency-friendly

## 📦 npm Package
Install via:
```bash
npm install @skynoveau/editor
