# Word to HTML Editor | Online Tool + npm Package

Easily convert Microsoft Word (.docx) documents into clean, semantic HTML.

This project offers:
- 🌐 A **browser-based editor** to paste or upload Word content and get instant HTML output.
- 📦 A **Node.js npm package** to integrate Word-to-HTML conversion into your own apps and pipelines.

## ✨ Features
- Converts DOCX (Microsoft Word) to HTML
- Removes unnecessary styles and Word-specific markup
- Supports basic formatting: headings, bold, italic, lists, tables, links, and images
- Clean, readable, semantic HTML output
- Lightweight and dependency-friendly

## 📦 npm Package
Install via:
```bash
npm install word-to-html-editor
